My Work

Most of the code I write professionally is private / under NDA, but here are a few samples of fun projects you can check out.

Teague

Craft CMS Website

Teague's website is a project I developed for Turnstyle, utilizing custom advanced animation sequencing through GSAP, as well as modern JavaScript page transitions built on top of BarbaJS.

This site is powered by Craft CMS, and utilizes modern tooling such as Webpack, with CI/CD to AWS servers through Forge.

Bloodworks Ventures

Craft CMS Website

The website for Bloodworks Ventures is a project I developed for Turnstyle, bringing a smaller site to life through subtle animations and speedy page transtions.

This site is powered by Craft CMS, and utilizes a wide variety of modern front end tooling including Webpack, ES6, BarbaJS, and Tailwind.

tapdat

Digital Tap List

tapdat is a digital tap list app, built to be displayed above a home or office kegerator. It currently supports coffee, cider, and beer.

I worked side-by-side with visual designers, originally from a Sketch file, then iterating in-browser. Using Contentful, the CMS is decoupled as an API, allowing the app to be extremely lightweight on the front end.
